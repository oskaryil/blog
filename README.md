# exp-hbs-passport-boilerplate

##Boilerplate includes:
* Express.js
* Handlebars.js
* Passport.js
* Gulp
* Sass
* Babel ES6
* Normalize.css
* Bootstrap v4
* Mongoose
* and other goodies!

##Included fonts
* Lato
* Roboto
* Knewave
* Ramabhadra
* Oswald
* Font Awesome

##Prequisities
* Node.js v4+
* MongoDB

##Install
```npm install ```

##Rename
```config.example.json to config.json```
###Linux users
```cp config.example.json config.json```

##Run
```npm start```

##Compile ES6
```npm run babel```

##Compile Sass
```gulp ```
