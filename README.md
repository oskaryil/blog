# exp-hbs-passport-boilerplate

##Boilerplate includes:
* Express.js
* Handlebars.js
* Passport.js
* Gulp
* Sass
* Babel ES6
* Normalize.css
* Bootstrap v4
* and other goodies!

##Included fonts
* Lato
* Roboto
* Knewave
* Ramabhadra
* Oswald
* Font Awesome

##Install
```npm install ```

##Run
```npm start```

##Compile ES6
```npm run babel```

##Compile Sass
```gulp```
